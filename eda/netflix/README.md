# Netflix EDA Project

## Overview
This project analyzes the Netflix Movies and TV Shows dataset.

## Analysis Included
- Data inspection (head, info, describe)
- Missing value handling
- Release year trend
- Year added trend
- Movie vs TV Show comparison
- Country distribution

## Key Findings
- Content increased significantly after 2015.
- Movies dominate compared to TV Shows.
- Peak content addition around 2019.
- United States is the largest content contributor.

## Tools Used
- pandas
- matplotlib
